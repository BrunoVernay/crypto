Sudoku-Löser

Mit diesem Plugin können die folgender drei Arten von Sudokus gelöst werden: Normale Sudokus, Killer-Sudokus und hexadezimales Sudokus.

Die folgende Hilfe dazu besteht aus 4 Abschnitten: Erklärung der Sudokus, Sudoku-Lösungstaktik, Bedienung des Plugins und Datei-Support.

Erklärung der Sudokus

Die normale Variante ist aus Zeitungen gut bekannt: Sie besteht aus einem 9x9-Quadrat, bei dem in jedes der 81 Felder eine Zahl zwischen 1 und 9 eingetragen werden muss. Jede der neun Zahlen muss so eingetragen werden, dass sie in jeder Zeile, jeder Spalte und in jedem 3x3-Teilquadrat (angezeigt durch die dickeren Linien) nur einmal vorkommt.

Ein hexadezimales Sudoku ist lediglich eine Erweiterung dieses Sudokus auf ein 16x16-Quadrat mit 4x4-Teilquadraten. Nun muss in jedes der 256 Felder eindeutig eines von 16 Zeichen (oft bezeichnet mit den Hexziffern 0 bis 9 und A bis F) eingetragen werden. Die Restriktionen sind entsprechend zur normalen Variante.

Die meisten betrachten das Killer-Sudoku als das schwierigste der drei Typen. Hierbei gibt es unterschiedliche Boxen mit mathematischen Operationen (Addition, Subtraktion, Multiplikation und Division). Wendet man die Operation auf alle Zahlen einer Box an, muss das in der Box angegebene Ergebnis entstehen. Wiederum sind die Zahlen pro Zeile, pro Spalte und pro "Rechen-Box" eindeutig. Es kann auch sein, dass zusätzlich die Zahlen in jedem 3x3-Teilquadrat nur einmal vorkommen dürfen. Das Plugin erlaubt, diese zusätzliche Teilquadrate-Regel (Box-Regel) ein- und auszuschalten.

Sudoku-Lösungstaktik

Nachdem Sudoku populär wurde, sind zahlreiche Taktiken zur Lösung online veröffentlicht worden. Einige dieser Taktiken sind in diesem Löser implementiert worden, um einem Benutzer zu helfen, ein Sudoku zu lösen. Für das normale und das Killer-Sudoku enthält dieses Plugin momentan keine Taktiken, denn diese können mit einem Brute-Force-Ansatz gelöst werden (Ausprobieren aller Möglichkeiten).

Das Killer-Sudoku lässt sich aufgrund der zusätzlichen Restriktionen deutlich schwieriger per Brute-Force lösen. Trotzdem sind auch dazu momentan keine Taktiken in das Plugin eingebaut, weil wir keine "guten" Taktiken fanden. Der eingebaute Brute-Force-Ansatz berechnet alle Möglichkeiten pro Einzelfeld unter Beachtung der mathematischen Beschränkungen (indem er mit der kleinsten Anzahl Möglichkeiten beginnt).

Das hexadezimale Sudoku ist wahrscheinlich am schwierigsten mit einem Brute-Force-Ansatz zu lösen aufgrund des vergrößerten Quadrats. Deshalb sind für diese Typ einige Taktiken eingebaut. Diese Taktiken werden für das normale Sudoku erklärt auf www.su-doku.net und www.sudokuoftheday.com. Die folgenden Taktiken sind hierfür verfügbar:

  • One possible (nicht erklärt)
  • Naked single
  • Hidden single
  • Block and Column
  • Naked subset
  • Candidate line
  • Double pair
  • Multiple lines

Bedienung des Plugins

Das Plugin hat zwei Modi: Eingabe und Lösen. Beim Start ist es im Eingabe-Modus, in dem man entweder ein existierendes Sudoku laden oder in dem man das neue Sudoku von Hand eintippen kann. Das Eingeben eines normalen oder hexadezimalen Sudokus ist unkompliziert: Die gegebenen Zahlen werden einfach in die richtigen Felder eingetragen.

Ein Killer-Sudoku einzugeben ist etwas komplizierter: Die Zahlen einzugeben ist wieder einfach, aber zusätzlich sind noch die Flächen (Rechen-Boxen), die mathematische Operation und das Ergebnis einzugeben. Dazu muss man nacheinander die Felder einer Rechen-Box anklicken, wodurch sie rot werden. Hat man alle Einzelfelder für eine Rechen-Box ausgewählt, wird der Operator aus dem Menü links gewählt und das erforderliche Ergebnis eingegeben. Es ist zu beachten, dass nur Einzelfelder, die nebeneinander liegen, in einer Rechen-Box liegen können.

Nachdem ein Sudoku geladen oder fertig eingegeben ist, muss man in den Lösen-Modus wechseln. Um das Sudoku dann zu lösen, ist als nächstes ein Einzelfeld anzuklicken und die entsprechende Zahl einzugeben. Das Plugin verhindert, dass man eine Zahl eingibt, die eine der gegebenen Regeln verletzt. Um es einfacher zu haben, kann das Plugin auch in allen Einzelfeldern alle möglichen Zahlen anzeigen. Wenn man nicht weiter weiß, kann man sich einen Hinweis geben lassen. Das Plugin wählt dazu ein leeres Einzelfeld zufällig aus und trägt die korrekte Zahl ein. Beachte, das der "Hinweis"-Button nicht sofort aufrufbar ist, wenn man in den Lösen-Modus wechselt, weil das Plugin zuerst das gesamte Sudoku im Hintergrund löst.

Mit dem Plugin kan man Sudokus nicht nur laden, sondern auch abspeichern, so dass man jederzeit unterbrechen kann.
Man kann sich auch Sudokus aus dem Internet herunterladen und diese in diesem JCT-Plugin öffnen: Diese müssen nur zu den jeweiligen Dateiformaten für die drei unterschiedlichen Sudoku-Typen passen (siehe Datei-Support).

Datei-Support

Das Plugin unterstützt unterschiedliche Dateiformate für die verschiedenen Sudoku-Typen. Im Folgenden wird erklärt, wie die Datei aussehen soll.

Für normale Sudokus ist das Format sehr flexibel: Das Plugin versucht, 81 unterscheidbare Zahlen zu finden (wobei die leeren Einzelfelder normalerweise als '0' oder '.' dargestellt werden).

Killer-Sudokus erfodern ein striktes Dateiformat: Alle Einzelfelder, die zu einer Rechenbox gehören, müssen dasselbe Zeichen (Character) enthalten. Diese Zeichen sind pro Rechenbox eindeutig. Die Zeichen müssen so angeordnet werden, wie sie auch im Sudoku stehen. Unterhalb des Sudokus ist dann für jedes Zeichen ein String anzugeben, der die Operation und das Ergebnis enthält, z.B. a x 24 or b + 12. Anstelle von '+' kann man als Operator auch '=' angeben. Ein komplettes Beispiel finden Sie unten.

Bei hexadezimalen Sudokus werden zwei Formate akzeptiert: Ein leeres Einzelfeld kann entweder per '.' oder per Leerzeichen gekennzeichnet werden. Im ersten Fall werden die Eingaben zeilenweise ohne Leerzeichen dazwischen erwartet. Im zweiten Fall werden die Eingaben auch zeilenweisen erwartet -- mit einem Leerzeichen zwischen jedem Zeichen. Welcher Typ benutzt wird, muss mit einer '1' oder '2' in der ersten Zeile der Datei angezeigt werden.

Das Klicken des "Autofill"-Buttons füllt ein leeres Einzelfeld, in dem nur eine Zahl möglich war. Das gefüllte Einzelfeld blinkt danach rot. Im Fall des hexadezimalen Sudokus wird durch Klick auf einen der Strategie-Buttons die entsprechende Strategie angewandt und die entspechenden Einträge vorgenommen. Die gewählte Strategie wird (ohne weitere Benutzereingabe) so oft wiederholt, bis damit keine weiteren leeren Einzelfelder mehr gefüllt werden können.
