Merkle Signature

This Virtualisation should show how the Merkle Signature Scheme works and is startable via the the menu Visualisation.

The Merkle Signature Scheme allows one to sign a message resistant against quantum computers.

Tab: Seed- and Key generation

On this page a seed must be chosen and secret Keys generated.

A seed can either be chosen by hand or can be randomly choosen by clicking the button.

Press the button to generate the key pairs.

Tab: MerkleTree

In this tab you can view the MerkleTree. It was generated by using the key pairs generated on the previous page. The lowest level are the leaves and at the top is the Merkle Root, also known as Merkle Tree Public Key.

One can click on one of the nodes to show the path to the root in yellow. If you click on one of the leaves also the verification path is shown in red which is path of the signature.

The number in the leaves coresponds to the index of the key pairs.

Tab: Signing

In this tab one can generate a signature using the keys generated in the first tab.

Here a random Text can be entered that will be signed

After clicking the button "Sign Message" in this field a signature for the message appears using the keys from the first tab. The notation is "Signature | Index of key pair | Authentication Path"
